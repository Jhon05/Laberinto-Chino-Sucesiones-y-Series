## Cambios de esta versión
- Nuevo tema visual: laberinto chino inspirado en la Gran Muralla, templos, linternas y dragones.
- Avatares cambiados a los 12 signos del zodiaco chino.
- Banco dinámico de preguntas reorientado a sucesiones, series y criterios usuales de convergencia.
- Pistas de retroalimentación orientadoras: ayudan a identificar el método sin revelar la respuesta antes de responder.
- Contraseña docente temporal oculta en la interfaz; el campo solo muestra caracteres enmascarados.
- Bloqueo docente ante salida de pantalla completa, Escape, cambio de ventana, clic derecho, pantallazo o atajos no permitidos.
- Informe final HTML tipo libro con fórmulas en LaTeX.

## Uso en GitHub Pages
1. Sube `index.html`, `styles.css`, `game.js` y este `README.md` a un repositorio.
2. Activa GitHub Pages desde la rama principal.
3. Abre la URL generada por GitHub Pages en Chrome o Edge.

## Recomendación para Brightspace
Para juegos con pantalla completa y reporte final, es mejor abrir el recurso en una pestaña nueva en lugar de incrustarlo dentro de la página.
